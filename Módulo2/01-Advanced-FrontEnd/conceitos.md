# Advanced FrontEnd

Projeto: https://github.com/Edugital/loja-ang-demo-descomplica/tree/main/src/app/features

## Configurações Projeto Angular

Iniciando o projeto:

```
npm install -g @angular/cli
ng version
ng new PROJETO
```

Gerando módulos e componentes:

```
ng generate module shared
ng generate component features/not-found
```
